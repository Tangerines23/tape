# 프레젠테이션 색상 코드 목록

## 주요 색상 팔레트

### Teal (청록색) - 강조 색상
- `teal-300`: `#5eead4` - 제목, 강조 텍스트
- `teal-400`: `#2dd4bf` - 구분선, 테두리
- `teal-500`: `#14b8a6` - 테두리 (투명도 포함)
- `teal-900/30`: `rgba(19, 78, 74, 0.3)` - 배경 강조 박스

### Slate (회색) - 기본 색상
- `slate-200`: `#e2e8f0` - 일반 텍스트
- `slate-300`: `#cbd5e1` - 보조 텍스트
- `slate-400`: `#94a3b8` - 플레이스홀더 텍스트
- `slate-500`: `#64748b` - 비활성 텍스트
- `slate-600`: `#475569` - 테두리
- `slate-700`: `#334155` - 배경, 테두리
- `slate-800`: `#1e293b` - 카드 배경
- `slate-900`: `#0f172a` - 메인 배경
- `slate-900/50`: `rgba(15, 23, 42, 0.5)` - 반투명 배경

### Gray (회색) - 배경 색상
- `gray-900`: `#111827` - 슬라이드 배경
- `gray-800`: `#1f2937` - 카드 배경 (일부)
- `gray-700`: `#374151` - 테두리, 호버 배경
- `gray-600`: `#4b5563` - 스크롤바, 테두리
- `gray-500`: `#6b7280` - 스크롤바 호버
- `gray-400`: `#9ca3af` - 보조 텍스트
- `gray-300`: `#d1d5db` - 텍스트
- `gray-100`: `#f3f4f6` - 밝은 텍스트

### Red (빨간색) - 경고/문제점 표시
- `red-900/30`: `rgba(127, 29, 29, 0.3)` - 경고 배경
- `red-500/50`: `rgba(239, 68, 68, 0.5)` - 경고 테두리
- `red-300`: `#fca5a5` - 경고 텍스트

### Black (검정색)
- `black`: `#000000` - 첫 슬라이드 배경
- `black/50`: `rgba(0, 0, 0, 0.5)` - 모달 오버레이
- `black/30`: `rgba(0, 0, 0, 0.3)` - 네비게이션 배경
- `black/20`: `rgba(0, 0, 0, 0.2)` - 썸네일 스트립 배경

### White (흰색)
- `white`: `#ffffff` - 기본 텍스트
- `white/80`: `rgba(255, 255, 255, 0.8)` - 아이콘, 텍스트
- `white/20`: `rgba(255, 255, 255, 0.2)` - 호버 배경
- `white/10`: `rgba(255, 255, 255, 0.1)` - 버튼 배경

## Tailwind CSS 클래스 매핑

### 배경색 (Background)
- `bg-slate-900` → `#0f172a`
- `bg-slate-800` → `#1e293b`
- `bg-slate-700` → `#334155`
- `bg-gray-900` → `#111827`
- `bg-gray-800` → `#1f2937`
- `bg-teal-900/30` → `rgba(19, 78, 74, 0.3)`
- `bg-red-900/30` → `rgba(127, 29, 29, 0.3)`
- `bg-black/50` → `rgba(0, 0, 0, 0.5)`

### 텍스트 색상 (Text)
- `text-white` → `#ffffff`
- `text-slate-200` → `#e2e8f0`
- `text-slate-300` → `#cbd5e1`
- `text-slate-400` → `#94a3b8`
- `text-teal-300` → `#5eead4`
- `text-teal-400` → `#2dd4bf`
- `text-red-300` → `#fca5a5`

### 테두리 색상 (Border)
- `border-slate-600` → `#475569`
- `border-slate-700` → `#334155`
- `border-teal-400` → `#2dd4bf`
- `border-teal-500/50` → `rgba(20, 184, 166, 0.5)`
- `border-red-500/50` → `rgba(239, 68, 68, 0.5)`

## 사용 예시

### 주요 강조 색상
- **Teal-400** (`#2dd4bf`): 주요 강조, 구분선, 활성 상태
- **Teal-300** (`#5eead4`): 제목, 부제목

### 배경 계층
1. **Slate-900** (`#0f172a`): 최상위 배경
2. **Slate-800** (`#1e293b`): 카드/컨테이너 배경
3. **Slate-700** (`#334155`): 중첩된 요소 배경

### 텍스트 계층
1. **White** (`#ffffff`): 최상위 텍스트
2. **Slate-200** (`#e2e8f0`): 일반 텍스트
3. **Slate-300** (`#cbd5e1`): 보조 텍스트
4. **Slate-400** (`#94a3b8`): 비활성/플레이스홀더 텍스트
