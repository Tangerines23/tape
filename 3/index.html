1. 개발 배경 및 필요성

산업적 배경: 전기차 부스바(Busbar) 절연 공정

  • 공정의 핵심: 배터리 부스바의 절연을 위해 테이프를 고속으로 감는 '테이핑 헤드(Taping Head)'의 정밀도가 품질을 좌우함.
  • 헤드의 특징: 고속 회전(High RPM)을 하는 정밀 구동체로, 지속적인 회전 부하로 인해 주기적인 보수와 교체가 필요한 핵심 관리 부품.

현장의 문제점 (Pain Point): "라인을 세워야만 가능한 검증"

헤드 수리나 교체 후 성능을 검증할 방법이 '실제 생산 장비 장착'밖에 없음.

[현재의 비효율적 프로세스]

  1. 생산 라인을 멈추고(Line Stop), 로봇에 헤드를 장착.
  2. 테스트 구동 중 미세한 밸런스 붕괴(소음/진동) 발견 시 즉시 불량 판정.
  3. 다시 탈거 → 분해/재조립 → 재장착 (무한 반복).

결과: 헤드 교체 시마다 라인 가동 중단(Downtime)이 발생하며 엔지니어의 공수가 낭비됨.

2. 프로젝트 목표

핵심 목표

"테이핑 헤드 성능 검증 시스템 (Test Jig) 개발"

생산 라인 로봇에 장착하기 전(Before), 별도의 공간에서 헤드의 회전 성능을 완벽하게 검증하는 표준화된 테스트 장비 개발.

🔧

유연성

다양한 규격의 헤드를 장착하여 테스트할 수 있는 구조.

🎯

정밀 제어

실제 로봇 구동 조건(RPM, 가감속)을 모사하여 필드와 동일한 환경 구현.

📊

데이터화

작업자의 '감'이 아닌 정량적 데이터(부하율, 진동 여부) 기반의 품질 표준 확립.

3.2 핵심 기술 선정: 왜 'EtherCAT 서보 제어'인가?

기존 방식(Pulse)의 한계

  • • 배선이 복잡하고 노이즈에 취약
  • • 고속 회전 시 실시간 데이터 피드백이 지연됨

EtherCAT 통신의 장점

실시간성

100Mbps급 고속 통신으로 지령 위치와 현재 위치를 실시간 동기화.

정밀 진단

서보 드라이버의 내부 데이터(전류치, 에러 코드 등)를 PLC로 즉시 전송하여 정밀한 부하율 모니터링 가능.

결론: 미세한 헤드 불량을 잡아내기 위해 LS ELECTRIC의 고성능 EtherCAT 모션 제어 솔루션을 채택함.

7. 프로젝트 수행 결과 요약

자재(BOM) 및 예산 활용

기업 지원: LS ELECTRIC의 XGT PLC 및 EtherCAT 서보 시스템 등 현업에서 사용되는 고사양 자재를 지원받아 구축.

문제 해결 (Problem Solving)

직면했던 문제:

초기 EtherCAT 통신 설정 시 축 동기화 오류 발생.

해결 과정:

네트워크 설정(XML 파일) 점검 및 PLC 모션 모듈의 파라미터 최적화를 통해 2축 동기 제어 안정화 성공.

8. 결론 및 기대 효과

기술적 성과

단순 시퀀스 제어를 넘어, EtherCAT 기반의 네트워크 모션 제어 기술을 습득하고 실제 시스템에 구현함.

현장 기여도

어떤 헤드든 '사전 검증(Pre-test)'을 표준화하여, 라인 가동 효율 극대화 및 유지보수 편의성 증대.
